Conquer ▼

Description: Conquer is a social media platform that aims to provide a supportive community designed to help individuals overcome various addictions by providing resources, tools, and a community to connect with. The world stigmatizes mental health making it difficult for people to seek help. We provide a safe space for users to join groups to chat with others. There is also a place to book with instructors and find a help center. This project is a concept of a social media platform advocated to mental health and self improvement. This is only a prototype and a Minimally Viable Product.

Features: Messaging, Find a help center, Book an instructor, Create a Status, Find Resources with dedicated links.

Tech Stack:

  • Frontend: React.JS & CSS
  • Backend: Node.JS, Express.JS, Firebase Authentication
  • Database: Firebase Firestore
  • Testing: Node.Js
  • API: Quotable, Leaflet + Open Road + Nominatim, Stripe
  • Deployment: Firebase Functions

Contributors: Amimul Arnab, Mohammed Rahman, Joshua Henry

Project Status: Prototype Completed.
